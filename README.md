# Tizen.NET.Sdk

Contains MSBuild targets and tools to enable the building of Tizen .NET Core projects.

## How to build this repo
1. Windows > Start > Developer Command Prompt for VS 2017
1. git clone 
1. Go to repository root 
1. Run build.cmd
